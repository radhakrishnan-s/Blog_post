Welcome to the Bloggers for Life!!!!
Here is a simple walk through of the site
1.New blog content can be added through the Add new blog link
2.The create new blog window have fields like Title, Category, Hash tag, Content.
3.After saving the content the blogs will get added to the dashboard.
4.If you want to have a look on the existing content, you need to click on the particular content and the content will be displayed.

If you have any queries please contact the support and Happy blogging.
